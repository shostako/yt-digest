# yt-digest

## 概要
YouTube動画の字幕を取得し、Gemini APIで詳細な解説を生成するWebアプリ

## 技術スタック
- **フロントエンド**: Next.js (App Router)
- **バックエンド**: Python FastAPI
- **字幕取得**: youtube-transcript-api
- **LLM**: Google Gemini API
- **デプロイ**: Render（フロント・API両方）
